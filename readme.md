# robertgabriel.ninja website  [![Build and Deploy](https://github.com/RobertJGabriel/Website/actions/workflows/build.yml/badge.svg)](https://github.com/RobertJGabriel/Website/actions/workflows/build.yml) <span class="badge-patreon"><a href="https://www.patreon.com/robertjgabriel" title="Donate to this project using Patreon"><img src="https://img.shields.io/badge/patreon-donate-yellow.svg" alt="Patreon donate button" /></a></span>
> Personal pwa website for robertgabriel.ninja. Feel free to use the set. 98% in lighthouse


## Usage

```js
    // Run dev mode
    yarn dev
    // Push live
    yarn fuckit
    // Build and push live
    yarn publish
```
