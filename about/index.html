My Story

Software engineer from Ireland. I build things for the web and care a lot about accessibility.

Hey, I'm Robert. I'm a software engineer from Cork, Ireland. I've been writing code since I was a teenager and it's pretty much all I've wanted to do since.

I'm dyslexic, and that's had a huge influence on what I work on. I built Helperbird because I wanted a tool that actually helped people like me use the web without struggling. It started as a side project and grew into something used by over a million people.

Along the way, I was lucky enough to be named on the Forbes 30 Under 30 list, which was honestly pretty surreal. I also enjoy speaking at conferences and meetups when I get the chance.

Outside of code, I drink too much coffee and I write about things that interest me — mostly tech, accessibility, and whatever else is on my mind.

Right now I'm running Helperbird and Coffee & Fun LLC, focused on building software that's useful, accessible, and hopefully a bit fun too.
